To add a file:
1. git add FILE/PATH/AND.extention
2. git commit -m 'COMMENT'
3. git push -u origin master

To remove a file:
1. git rm FILE/PATH/AND.extention
2. git commit -m 'COMMENT'
3. git push -u origin master

To update a file:
1. git add FILE/PATH/AND.extention
2. git commit -m 'COMMENT'
3. git push -u origin master
